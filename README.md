# jtlv

Simple library that parses BER-TLV (as String or byte array) into a TLV structure for easy handling

## Getting Started

### Dependencies

* Requires Java 17 LTS.

### Installing

* Clone the repo
```
git clone
```
* Generate the jar package with maven
```
mvn package
```
* Import the jar to your project
